A brief Guide to SEO and HTML Best Practices

How Search Engines Work

Search engines used automated programs called "crawlers" to comb through billions of web page documents through their links and record the data from their metatags in huge servers. When a user enters a search query into the engine, results are rendered based on relevance of the user's search to results, and popularity. This is how a search engine like google creates an order with which to display results.

Source: The Beginner's Guide to SEO

The Importance of Keywords and Descriptions as Metatags

Metatags are a shortened summary or set of keywords that, when given to a user, will determine whether or not a particular site will be entered or not, based on the user's needs. Metatags like the HTML title tag and meta descriptions are what a search engine will show a user when the site is returned as a result, with the title being what the search engine returns as the site's title, and meta description being the little blurb under the title. Meta keywords are another element used by the search engine to find the site based on a user's search.

Source: conductor.com
